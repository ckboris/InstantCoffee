Setup

Configuring your system for Java EE.

Install JDK

Setup JAVA_HOME in Windows. Follow the steps laid out here. Note that you should use an absolute path name to your Java installation, e.g. C:\Program Files\Java\jdk1.8.0_144. Use JAVA_HOME\bin in your Path. Note: if this doesn't work, look at the directory C:\ProgramData\Oracle\Java\javapath. If there are symbolic links in here that point to the wrong Java version (e.g. Java 7), they may need to be deleted.

Install Oracle WebLogic Server 12c

  1. Launch the command prompt as administrator. Navigate to the location where you extracted the WebLogic Server files.
  2. Type java -jar fmw_12.2.1.3.0_wls_quick.jar (or whatever version number of WebLogic you downloaded). If the installer doesn't recognize your JAVA_HOME, follow the instructions here.
  3. NOTE: wls12213 (or something else starting with wls) is your ORACLE_HOME. ORACLE_HOME by default is placed wherever you install WebLogic.
  4. Travel to ORACLE_HOME\oracle_common\common\bin and run config.cmd.
  5. During the installation, take note of the default domain location (which will be something like C:\wls12213\user_projects\domains\base_domain), the admin name and password, and admin server URL.

Install NetBeans

Once NetBeans is installed, add a new server under Services -> Servers -> Add Server. Choose WebLogic when prompted. Select your ORACLE_HOME directory as the server location.

Install Oracle Database

Take note of your Global Database name, username, password, and pluggable database name. Note: if the path name is too long, uncheck the box in the upper right hand corner, as seen in this answer. Log into https://localhost:5500/em/login as sys (check as sysdba) and whatever password you specified during installation. If this doesn't work, open SQL Plus, enter username sys as SYSDBA and the password you specified during installation.

Download Oracle SQL Developer

This doesn't need to be installed.
